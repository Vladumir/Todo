# TodoDesktop

**TodoDesktop** — это настольное приложение для управления личными задачами, разработанное на **WPF (C#)** с использованием **.NET 6**. 

Приложение помогает пользователям организовывать свой день, создавать задачи с категориями и сроками выполнения, а также отслеживать историю выполненных дел.

## Функциональность

*   **Авторизация и Регистрация:**
    *   Вход в систему по Email и паролю.
    *   Регистрация новых пользователей с валидацией данных.
*   **Управление задачами:**
    *   Создание задач с указанием названия, описания, категории, даты и времени.
    *   Просмотр списка активных задач.
    *   Отметка задач как "Выполненные".
    *   Удаление задач.
*   **Категории:** Разделение задач по типам (Дом, Работа, Учеба, Отдых).
*   **История:** Отдельный просмотр завершенных задач.
*   **Профиль пользователя:** Отображение имени текущего пользователя.

## Технологический стек

*   **Платформа:** .NET 6.0 Windows
*   **UI Фреймворк:** WPF (Windows Presentation Foundation)
*   **Язык:** C#
*   **Архитектура:** Single Window Application (Навигация на основе страниц `Page`)
*   **Паттерны:**
    *   Repository Pattern (для работы с данными).
    *   Data Binding (привязка данных в XAML).

## Структура проекта

Проект был рефакторизован для использования навигационной структуры (Page-based navigation).

```text
TodoDesktop/
│
├── Desktop/                  # Основной проект WPF
│   ├── App.xaml              # Точка входа, ресурсы стилей
│   ├── MainWindow.xaml       # Главное окно-контейнер (Shell) с Frame
│   │
│   ├── View/                 # Страницы приложения (UI)
│   │   ├── LoginPage.xaml        # Страница входа
│   │   ├── RegistrationPage.xaml # Страница регистрации
│   │   ├── TasksPage.xaml        # Главный список задач
│   │   ├── AddTaskPage.xaml      # Форма создания задачи
│   │   ├── HistoryPage.xaml      # История выполненных задач
│   │   └── MainEmptyPage.xaml    # Страница-заглушка (если нет задач)
│   │
│   ├── Repository/           # Логика работы с данными
│   │   ├── TaskRepository.cs
│   │   └── UserRepository.cs
│   │
│   ├── images/               # Ресурсы (логотипы, аватары)
│   └── InputValidator.cs     # Валидация ввода
│
└── Todo.Entities/            # Библиотека классов (Модели)
    ├── TaskModel.cs          # Модель задачи
    └── UserModel.cs          # Модель пользователя
